# my-butler

Butler in my slack

## Use
- Nature Remo
- IFTTT
- AWS CLI, CDK
- TypeScript, Go

## Setup

```shell
$ npm install @aws-cdk/aws-events @aws-cdk/aws-events-targets \
 @aws-cdk/aws-iam @aws-cdk/aws-lambda
$ npm run build
$ cdk deploy
```
